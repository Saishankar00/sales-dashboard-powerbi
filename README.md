# 📊 Sales Dashboard for Regional Performance

## 📌 Overview
This project analyzes regional sales performance using **SQL** and **Power BI**. The dashboard provides interactive insights into sales trends, regional performance, product categories, and business KPIs to support data-driven decision-making.

## 🚀 Tech Stack
- Power BI
- SQL (MySQL)
- Microsoft Excel

## 📂 Project Files
- 📄 sales project.pbix
- 📄 Sales_Dashboard_Dataset.xlsx
- 📄 sales_analysis.sql

## 📈 Dashboard Features
- Sales Overview
- Regional Performance
- Product Analysis
- Monthly Sales Trends
- KPI Cards
- Interactive Filters

## 🎯 Skills Demonstrated
- SQL Joins
- Data Cleaning
- Data Transformation
- Power Query
- DAX
- Data Visualization
- Business Intelligence
